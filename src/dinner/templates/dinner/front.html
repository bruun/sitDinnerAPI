SiT Kafe API

FAQ

  1. Where do you get the data from?

    All menus are found here: http://sit.no/content/31542/Middag

  2. Is this API free?

    This API is free for non-commercial use.

  3. How do I use the API?

    There are several ways to access the data:

    • List cafeterias: http://sit.tbruun.bo/api/cafeterias/
    • Meal for day in cafeteria: http://sit.tbruun.bo/api/cafeteria/NAVN/YYYY-MM-DD
    • Meal from day in week: http://sit.tbruun.bo/api/cafeteria/NAVN/weekly/YYYY-MM-DD

  4. The API says no meal was found on a given weekday, but the SiT website claims there is.

    The SiT RSS-feeds change from time to time, making this site unable to process them.
    If you discover such a case, please send me an email at thomas(dot)bruun(at)gmail(dot)com, with information about the cafeteria and date.

  5. Can I access menus from previous weeks?

    Yes, if it was accessed via the API that week. Unfortunately SiT does not keep a record of meals from previous weeks.

  6. I am having problems with the iPhone app.

    I the app claims that no meal was found, please refer to question 4.
    For other issues, please send me an email at thomas(dot)bruun(at)gmail(dot)com, with information with your iPhone model, iOS version, and an explanation of the issue

  7. Who are you?

    My name is Thomas Bruun, and I study computer science at the Norwegian University of Science and Technology (NUTS!).

  8. What is the API written in?

    Django!
